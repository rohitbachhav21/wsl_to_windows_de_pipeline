Always run Spark with the Kafka connector package for Apache Kafka:

spark-submit --packages org.postgresql:postgresql:42.6.0,mysql:mysql-connector-java:8.0.33,org.apache.spark:spark-sql-kafka-0-10_2.12:3.5.0 spark_kafka_consumer.py 2&>1 | tee spark_debug.log

The first run will download the connector.
